# Công bố lên Hugging Face Spaces

1. Đăng nhập https://huggingface.co/spaces
2. Chọn **Create new Space**.
3. Đặt tên ví dụ: `cvnss40-trace-learning-pro-v4`.
4. Chọn SDK: **Static HTML**.
5. Upload toàn bộ file trong thư mục này vào root của Space.
6. Commit. Space sẽ tự phục vụ `index.html`.

Nếu cập nhật phiên bản, chỉ cần thay file `index.html` và `cvnss40_converter_pro_v4_trace.js`, sau đó commit lại.
